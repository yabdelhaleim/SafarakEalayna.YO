# موديول فوري - دليل الاستخدام الشامل 🚀

## 1️⃣ واجهة فيليمنت (لوحة التحكم)

### 🎯 الوظيفة

يمكن للمسؤول من خلال فيليمنت إعداد وضبط جميع بيانات موديول فوري.

### 📊 الإعدادات المتاحة

#### أ) أنواع العمليات (`FawryOperationType`)

**المسار:** Filament Admin → إعدادات فوري → أنواع عمليات فوري

**الحقول المتاحة:**
- **الكود (code):** معرف فريد مثل `withdrawal`, `deposit`
- **الاسم (عربي/إنجليزي):** الاسم المعروض للمستخدم
- **اللون:** لون مميز للعرض في الواجهة
- **الأيقونة:** أيقونة Heroicons
- **الوصف:** شرح تفصيلي لنوع العملية
- **نشط:** تفعيل/تعطيل نوع العملية
- **الترتيب:** ترتيب العرض في القوائم

**الأنواع الافتراضية:**
- سحب (Withdrawal)
- إيداع (Deposit)
- سداد (Payment)
- تصريح سفر (Travel Permit)

#### ب) طرق الدفع (`FawryPaymentMethod`)

**المسار:** Filament Admin → إعدادات فوري → طرق دفع فوري

**الحقول المتاحة:**
- **الكود (code):** معرف فريد مثل `cash`, `bank_transfer`
- **الاسم (عربي/إنجليزي):** الاسم المعروض
- **اللون:** لون مميز
- **الأيقونة:** أيقونة Heroicons
- **اسم المزود:** مثل (فوري، كاش، بريد)
- **اسم البنك:** اسم البنك للتحويلات
- **اسم الفرع:** فرع البنك
- **رقم الحساب:** رقم الحساب البنكي
- **رقم المحفظة:** رقم المحفظة الإلكترونية
- **بيانات إضافية (metadata):** بيانات JSON إضافية
- **الحساب الافتراضي:** الحساب المالي المرتبط
- **نشط:** تفعيل/تعطيل
- **الترتيب:** ترتيب العرض

**الطرق الافتراضية:**
- نقدي (Cash)
- تحويل بنكي (Bank Transfer)
- محفظة كاش (Cash Wallet)
- خزينة المكتب (Office Safe)
- درج المكتب (Office Drawer)

#### ج) العملات (`FawryCurrency`)

**المسار:** Filament Admin → عملات → إعدادات العملات

**الإعدادات الخاصة بفوري:**
- **سعر الصرف:** سعر الصرف الخاص بفوري
- **الحد الأدنى:** أقل مبلغ مسموح
- **الحد الأقصى:** أكبر مبلغ مسموح
- **نسبة الرسم:** رسم بنسبة مئوية
- **رسم ثابت:** رسم ثابت لكل معاملة

## 2️⃣ واجهة Vue (واجهة التشغيل)

### 🎯 الوظيفة

واجهة التشغيل للموظفين لتنفيذ معاملات فوري اليومية.

### 📝 خطوات تنفيذ معاملة

#### الخطوة 1: بيانات العميل
```
1. اختيار العميل من القائمة
   - قائمة العملاء المسجلين في النظام
   - أو إضافة عميل جديد

2. إدخال رقم المرجع (اختياري)
   - رقم العملية أو مرجع العميل
```

#### الخطوة 2: تفاصيل العملية
```
1. اختيار نوع العملية
   - القائمة تُجلب من إعدادات فيليمنت
   - الأنواع النشطة فقط تظهر

2. اختيار العملة
   - العملات المتاحة من إعدادات فيليمنت
   - سعر الصرف يُحتسب تلقائياً
```

#### الخطوة 3: التسعير
```
1. مبلغ العميل
   - المبلغ الذي يريده العميل

2. سعر فوري
   - التكلفة من فوري

3. سعر البيع
   - السعر المباع للعميل

⚡ الربح يُحتسب تلقائياً:
   الربح = سعر البيع - سعر فوري
```

#### الخطوة 4: بيانات الدفع
```
1. اختيار طريقة الدفع
   - الطرق المُعدة في فيليمنت
   - تفاصيل كل طريقة تظهر تلقائياً

2. إدخال المبلغ المدفوع
   - المبلغ الذي استُلم من العميل
```

#### الخطوة 5: حفظ المعاملة
```
عند الضغط على "إنشاء المعاملة":
✓ تُحفظ في سجل فوري
✓ تُسجل في الحسابات المالية
✓ تُنعكس على تقارير الأرباح
✓ تُحدَّث الخزينة المختارة
```

```sql

- client_name
- operation_type

- client_amount
- fawry_price
- selling_price
- profit (يُحتسب تلقائياً)

- account_id (FK → accounts)
- payment_method
- amount
- reference_number
- notes
- payment_details (JSON)
- expense_transaction_id (FK → transactions)
- income_transaction_id (FK → transactions)
- created_at, updated_at, deleted_at
```

```sql
- id
- code (unique)
- name_ar, name_en
- color
- icon
- description_ar, description_en
- is_active
- order
- created_at, updated_at, deleted_at
```

```sql
- id
- code (unique)
- name_ar, name_en
- color
- icon
- description_ar, description_en
- provider_name
- bank_name
- branch_name
- account_number
- phone_number
- metadata (JSON)
```

## 🔧 الصيانة والتطوير

### إضافة نوع عملية جديد

1. اذهب إلى Filament Admin → إعدادات فوري → أنواع عمليات فوري
2. اضغط "إنشاء جديد"
3. املأ البيانات:
   - الكود: `new_operation`
   - الاسم: "العملية الجديدة"
   - اللون والأيقونة
4. احفظ
5. سيظهر فوراً في واجهة Vue

### إضافة طريقة دفع جديدة

1. اذهب إلى Filament Admin → إعدادات فوري → طرق دفع فوري
2. اضغط "إنشاء جديد"
3. املأ البيانات:
   - الكود: `new_method`
   - الاسم: "طريقة جديدة"
   - اسم المزود: "اسم الجهة"
   - رقم الحساب/المحفظة
   - الحساب المالي الافتراضي
4. احفظ
5. سيظهر فوراً في واجهة Vue مع التفاصيل
