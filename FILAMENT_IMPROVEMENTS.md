## الاستخدام

### تفعيل التحسينات
1. تأكد من تحديث ملفات CSS:
   ```bash
   npm run dev
   ```

2. مسح ذاكرة التخزين المؤقت:
   ```bash
   php artisan cache:clear
   php artisan view:clear
   php artisan config:clear
   ```

3. إعادة تشغيل الخادم:
   ```bash
   php artisan serve
   ```

### التخصيص
يمكنك تخصيص الألوان والمكونات من خلال:
- `app/Filament/AdminPanel.php` - إعدادات اللوحة الرئيسية
- `resources/css/filament/admin/theme.css` - الأنماط الرئيسية
- `resources/css/filament/admin/resources.css` - أنماط الموارد
